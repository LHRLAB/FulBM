FulBM: Batch maintenance for 3-hop cover label on large-scale complex networks

## Prepare

- install cmake 3.14+

## Usage

    .../FulBM$ mkdir build
    .../FulBM$ cd build
    .../FulBM/build$ cmake ..
    .../FulBM/build$ make
    .../FulBM/build$ ./correctness_test
